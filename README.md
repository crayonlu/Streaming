# Streaming

跨平台直播播放器，支持斗鱼、B站双平台。

![demo](assets/demo.png)

## 功能

- 多平台直播：斗鱼、B站
- 直播回放：斗鱼全量录像、B站授权剪辑
- 画质选择：多个 CDN 路线自动择优
- 暗色主题

## 安装

从 [Releases](https://github.com/crayon/Streaming/releases) 下载对应系统的安装包。

## 注意事项

- 平台接口可能有访问频率限制，过于频繁的请求会触发验证码校验
- 本项目仅供学习编程目的使用，未进行任何逆向工程
- 所有直播内容（视频、音频、图像等）**版权归属原平台所有**，本软件仅作技术聚合与转码展示
- 如有任何问题请与开发者联系，如有侵权请联系开发者删除
